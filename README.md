# SocialzApp
- A Jetpack compose SocialzApp consuming REST Posts Api to display a list of available posts

## Tech Stack
- Jetpack compose
- MVVM
- Retrofit
- Coil library for Image Loading 
- Hilt [Dagger-Hilt]
- Coroutines [Asynchronous Programming]
